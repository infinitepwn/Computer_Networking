# 定义
Code Divsion Multiplexing
当码分复用信道被多个不同地址的用户共享时，就称为码分多址CDMA

每一个用户可以再同样的时间使用同样的频带进行通信

## 如何发送信息
在CDMA中，每一个比特时间划分成m个短的间隔，称为码片chip，使用CDMA的每一站被指派一个唯一的m bit码片序列，如果要发送比特1，就发送这个码片序列，如果要发送比特0，就发送二进制反码，通常把0记作-1，1记作+1

这使得数据率要扩大到原来的m倍，这称为扩频

## 正交
给每一站分配的码片序列各不相同，且必须相互正交，
### 规格化内积
$$S\cdot T = \frac{1}{m} \sum_{i=1}^mS_iT_i=0$$

### 两个性质
- S和各站反码也正交（加个负号就行）
- S和自己的规格化内积都是1，（m个1除以m）
