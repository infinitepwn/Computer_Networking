# 网络层

## 数据层面

进行数据转发

IPv4 地址 = 32位二进制，通常用**点分十进制**表示，如 192.168.1.100

IP 地址 = **网络号（Net ID）** + **主机号（Host ID）**

有两种划分：分类和无类别(CIDR)

分类可以迅速得到主机号

CIDR可以实现按需分配，但需要通过子网掩码得到网络地址

由于路由聚合有可能导致匹配到多个网络地址，此时选择最长前缀原则进行匹配

如何加速匹配效率



默认路由0.0.0.0/0 所有都可以匹配成功,放在路由表的底端

主机路由A.B.C.D/32 频繁访问的可以直接存主机路由，放在顶端
